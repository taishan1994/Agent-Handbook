# Chapter 1: Prompt Chaining with PocketFlow

## 概述

本示例展示了如何使用PocketFlow框架实现**提示词链（Prompt Chaining）**模式。提示词链是将复杂任务分解为多个顺序执行的子任务，每个子任务通过专门的提示词处理，前一个任务的输出作为后一个任务的输入。

## 实现功能

本示例实现了一个技术规格提取和转换的流程：

1. **提取技术规格**：从文本中提取处理器、内存、存储等信息
2. **转换为JSON格式**：将提取的信息转换为结构化的JSON对象
3. **验证JSON格式**：确保输出的JSON格式正确有效
