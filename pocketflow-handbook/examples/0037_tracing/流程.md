# PocketFlow Tracing 项目使用指南
## 项目介绍
这是一个为PocketFlow工作流提供全面可观测性的工具，使用Langfuse作为追踪后端。通过简单地添加装饰器，您可以自动追踪PocketFlow工作流中的所有节点执行、输入、输出和错误。

## 主要功能
- 自动追踪：通过单个装饰器追踪整个流程
- 节点级可观测性：自动追踪每个节点的prep、exec和post阶段
- 输入/输出跟踪：捕获工作流中流动的所有数据
- 错误跟踪：自动捕获和追踪异常
- 异步支持：完全支持AsyncFlow和AsyncNode
- 最少代码更改：只需在flow类上添加@trace_flow()装饰器
- Langfuse集成：利用Langfuse强大的可观测性平台
## 使用步骤
### 1. 安装依赖
```bash
pip install -r requirements.txt
```
### 2. 配置Langfuse
- 复制 `.env.template` 到 `.env`
- 替换 `LANGFUSE_SECRET_KEY` 和 `LANGFUSE_PUBLIC_KEY` 为您的实际值
- 配置其他可选参数（如 `LANGFUSE_HOST`）
### 3. 启用追踪
- 在您的PocketFlow工作流类上添加 `@trace_flow()` 装饰器
- 确保您的工作流类继承自 `BaseFlow`
### 4. 查看追踪结果
- 运行您的工作流
- 在Langfuse控制台查看自动生成的追踪数据
  - 导航到 "Traces" 页面
  - 找到您的工作流运行记录
  - 点击记录查看详细的节点执行信息、输入输出数据、错误信息等
